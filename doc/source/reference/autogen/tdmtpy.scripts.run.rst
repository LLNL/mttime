tdmtpy.scripts.run
==================

.. note::

   This script automatically installs during the setup procedure with the
   name ``$ tdmtpy-run``. For more info on the command line options,
   please run ``$ tdmtpy-run --help``.
   Alternatively you can also execute ``$ python -m tdmtpy.scripts.run``.
